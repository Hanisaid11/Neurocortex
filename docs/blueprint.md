# **App Name**: NeuroCortex Pro

## Core Features:

- Knowledge Hub RAG: An AI-powered tool that searches indexed medical literature to provide cited, evidence-based answers to clinical questions.
- Diagnostic Imaging Workspace: A medical-grade UI featuring a DICOM viewer with windowing, zooming, and panning for MRIs and CT scans.
- Vision AI Diagnostic Tool: Analyze medical imagery and surgical video frames using generative vision models to detect clinical anomalies.
- OR-Ready Voice Assistant: Hands-free AI tool for surgical dictation and real-time medical inquiries while scrubbing or operating.
- Provider Orchestrator: Advanced settings to switch between Grok and Gemini, featuring frontend key validation specifically for 'AQ' prefixed Gemini tokens.
- Case Study Quizzer: An AI tool that generates board-style prep quizzes and flashcards based on textbooks in the Knowledge Hub.
- Clinical Outcomes Log: A secure clinical database for tracking and querying anonymized surgical case logs and research outcomes.

## Style Guidelines:

- Primary color: Clinical Cobalt (#2D68FF), providing a high-contrast, professional sense of trust and technological precision.
- Background: Midnight Obsidian (#0C0E12), a deeply desaturated version of the primary for a fatigue-reducing dark-mode workspace.
- Accent: Vivid Azure (#00D1FF), used for interactive DICOM elements and real-time surgical voice assistant status indicators.
- Font pairing: 'Space Grotesk' for precise, modern headlines and 'Inter' for high-readability medical text and literature.
- Code and data font: 'Source Code Pro' used specifically for surgical metrics and DICOM metadata values.
- Lucide icons using ultra-thin weights (1px to 1.5px) for a surgical, technical aesthetic throughout the UI.
- Fixed sidebar navigation for quick switching between critical clinical modules (Imaging, Research, Cases) on mobile and desktop.
- Micro-interactions on AI search inputs and pulse animations for the hands-free voice assistant listening state.
